A fully responsive, modern pizzeria website for Haga Pizzeria in Norrköping, Sweden.

✨ **Modern Design**
- Premium Italian aesthetic with warm colors
- Smooth animations and hover effects
- Professional but welcoming layout
- Fully responsive (mobile, tablet, desktop)

📱 **User Experience**
- Interactive menu with category tabs
- Sticky navigation bar
- Smooth scroll navigation

- Quick access to contact info

🎯 **Conversion Focused**
- Prominent call-to-action buttons

## 🚀 Quick Start

### View the Website
1. Open `index.html` in your browser
2. Or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

## 🎨 Customization

### Update Contact Information
Edit these values in `index.html`:
- Phone: `011-18 26 01`
- Address: `Breda vägen 28, 602 14 Norrköping`
- Opening hours: Update in Quick Info section

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-red: #c41e3a;
    --accent-orange: #ff6b35;
    /* ... other colors ... */
}
```
